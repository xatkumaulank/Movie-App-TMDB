# Movie-App-TMDB
Fetch data from the Internet with theMovieDB API

## Các kỹ thuật sử dụng
1. Android Architecture Components
2. Paging
3. Room
4. ViewModel
5. LiveData
6. Android Data Binding
7. Retrofit for REST api communication
8. Picasso for image loading
